# music

![效果图](https://images.gitee.com/uploads/images/2019/0413/185944_5b3a5fd7_3028814.png)

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
